This library contains a number of modules that can be used to support the CEG4166/CSI4141 projects. They are designed to
run in the AtMega 2560 Arduino card.

Section 1 USART Serial Module

This module allows the use of any of the 4 USARTs available in the microcontroller. A number of Unix/Linux like print functions are provided as well as low level character read and write functions.

Section 2 Servo Motor Motion Module

This module manipulates the microcontroller timer modules that allows control of any of the 3 servo motors available on the StingRay robot.
